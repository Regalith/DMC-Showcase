How it works

A community-driven question and answer platform targeted towards programmers.

Debug My Code is a fully comprehensive code debugging platform designed to give users all the tools necessary to both ask tough programming questions and answer them in more thorough ways compared standard question and answer sites. By packaging in a bounty system, instant messaging, forums, online compiling, and web-based remote desktop support into one responsive web app, we provide a faster and more personalized debugging experience compared to any other system online.

Ask

A user asks a question with an optional monetary bounty, which can be collected upon someone correctly answering it

Answer

Other users can reply to the question and directly leave answers, or subscribe to the question to offer direct assistance

Direct Assistance

Once users subscribe to a question, they are given a multitude of tools to help the asker, including online remote desktop support, in-site instant messaging, and online compiling

Features

A community-driven question and answer platform targeted towards programmers.

React

A fully responsive isometric web application with on Facebook's React.js. It's built in features allow for faster page loads and more dynamic web pages.

Guacamole RDP

HTML 5 Clientless Remote Desktop capabilities integrated right into the website. Users have the capability of directly assisting others with problems requiring a more direct approach, without ever leaving the website.

Online Compiling

Users can compile and edit code inside the webpage using the Ace web editor and our own cloud compilers.

Question and Answer System

A Standard answer hub system, but with extra features to seamlessly hook into the other support tools offered in the web application.

Question Bounties

Users can place monetary sums on questions to be collected by others upon a successful answer.

Instant Messaging and Push Notifications

Through the use of React and Socket.IO, users gain real time communication and notification tools for a more responsive webpage.
